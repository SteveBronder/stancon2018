Hows it Work? - Inverse

  • Calculate \(A1\) and \(A2\) inverse sequentially (\(C1\) and \(C2\)) \[ C3 = -C2 \times A3 \times C1 \]

  • Blocking algorithm by (Mahfoudhi, 2012)

Calculating A1 and A2 Inverse

  • Calculate \(A1_{A1}\) and \(A1_{A2}\) inverse sequentially (\(A1_{C1}\) and \(A1_{C2}\)) \[ A1_{C3} = A1_{-C2} \times A1_{A3} \times A_{C1} \]

Hows it Work? - Cholesky

  • Same as inverse, but only sequential for \(A_{11}\) \[L_{21} = A_{21} (L_{11}^T)^{(-1)} \; \;\;\; \; L_{22} = A_{22} - L_{21} (L_{21})^T\]

  • Blocking algorithm of (LouterNool, 1992)

Example: 1D GP Regresssion

  • Example from (Betancourt, 2017)
  • Relationship between \(x\) and \(y\) with added Gaussian noise:

\[x_i \sim_{\text{iid}} U(-10,10)\] \[y_i | x_i \sim_{\text{iid}} N \left( f(x), \frac{1}{10} \right), i = 1..n,\] where \[f(x) = \beta(x + x^2 - x^3 + 100 \sin 2x - \alpha)\]

  • Parameters \(\beta\) and \(\alpha\) s.t. \(E[f] = 0\) and \(Var[f] = 1\).

Code Example

  • Take Michael's code from his GP tutorial online
  • replace
matrix[N1, N1] L_K = cholesky_decompose(K);

with

matrix[N1, N1] L_K = cholesky_decompose_gpu(K);

Pass compiler options:

STAN_OPENCL=true
OPENCL_DEVICE_ID=0
OPENCL_PLATFORM_ID=0
